# Architecture Decision Records (ADR)

Use this folder for important architecture decisions that include trade-offs.

## Naming Convention

`NNNN-short-title.md` (example: `0001-api-error-envelope.md`)

## Suggested Template

```markdown
# NNNN - Decision Title

## Status
Accepted | Proposed | Superseded

## Context
What problem are we solving?

## Decision
What was decided?

## Consequences
Benefits, costs, and risks.
```
